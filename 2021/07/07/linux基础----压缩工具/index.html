    linux基础----压缩工具

    发布时间 : 2021-07-07 15:12

    1. 压缩工具

    压缩工具

    man compress 压缩算法 #目前已经没有使用了
    压缩
    compress filename #原来的文件删除了 压缩包默认是filename.Z
    compress -c filename >m.Z #压缩 文件不删除原文件，原理，将这些数据文件的信息重定向到文件
    解压
    uncompress filename.Z #删除原来的压缩 包，文件解压 出来
    预览
    zcat filename.Z

    gzip
    压缩
    gzip filename #压缩，原先的文件删除了
    #不删除文件并压缩 gzip -c filename > filename.gz
    压缩比率
    gzip -9 filename -c > filename.gz
    解压
    gunzip filename.gz #解压 并删除压缩 包
    预览
    zcat filename.gz

    将命令执行结果并打包
    cat messages |gzip > messages.log.gz

    命令提示符出现乱码，可以使用reset 重置当前终端 也可以退出重新登录

    lastb -f file1 |gzip >file1.gz

    lastb -f file1 |cut -d” “ -f1 |sort |uniq -c |sort -nr |less #取第一字段|排序|次数|按次数排序
    #lastb -f file1 |cut -d” “ -f3|sort|uniq -c|sort -nr |head #第3个字段 取前10
    lastb -f file1 |tr -s “ “ |cut -d” “ -f3 |sort|uniq -c |sort -nr|head

    bzip2
    压缩文件
    bzip2 -k filename #压缩 文件并保留原文件-k选项
    解压
    bunzip2 filename.bz2
    预览
    bzcat filename.bz2

    xz
    内核官网压缩 就是xz
    压缩
    xz -k file #保留原文件压缩 -k keep
    #压缩比强大，耗时
    解压
    unxz file.xz
    预览
    zxcat file.xz

    以上只能压缩文件

    压缩文件夹
    打包并压缩
    zip #和windows通用
    zip -r sysconfig /etc/sysconfig #打包压缩并命名sysconfig.zip
    解压
    zip -d sysconfig.zip
    unzip sysconfig.zip

    cat /var/log/messages|zip message - #message.zip

    tar 打包并压缩
    tape archive 磁带归档的缩写

    创建归档
    tar -cpvf /dir/file.tar. file #p保留属性 c是创建 v是过程显示详细信息 f后面跟文件
    追加文件至归档!!!!!!!注：不支持对压缩文件追加
    tar -r -f /file.tar file
    #预览
    tar tvf file.tar
    解包
    tar xvf file.tar
    tar xvf file.tar -C /mnt #解压到指定的目录下

    结合压缩工具
    -j bzip2 -z gzip -J xz

    tar zcvf file.tar.gz file/
    tar jcvf file tar.bz2 file/
    tar Jcvf file tar.xz file/

    解压
    tar xf tar.gz|tar.bz2|tar.xz

    查看内核代码多少行

    cat `find  -name "*.c" `|wc -l

    看内核代码是一个工程，一个人根本看不完
    可以针对性的去学其中一个模块

    将一个压缩包切割成若个小块
    split -b 大小 包名 包的前缀名
    split -b 5M name.tar.xz name_part

    合并压缩包
    cat name_part* > name.tar.xz

    cpio 文档
    file /boot/initramfs-xxx.img #cpio文件

    预览
    cat xxx.img |cpio -it
    cat -tv < xxx.cpio
    cat xxx |cpio -tv
    解包
    cpio -idv < xxx.cpio
    cat xxx | cpio -idv

    备份
    find -type f |cpio -ov > bak.cpio
    追加
    find type -f |cpio -oA -F bak.cpio
